# Benchmark des outils de littératie IA

## Objectif

Comparer les approches de formation/sensibilisation IA sans reproduire leur
contenu propriétaire.

## Méthode

Analyser uniquement des éléments publics : positionnement, formats, mécaniques
pédagogiques, transparence sources, limites visibles. Ne pas copier questions,
prompts, UX ou contenus propriétaires.

## Grille d'analyse

| Critère | Question |
| --- | --- |
| Pédagogie | scénario ou cours magistral ? |
| Sources | claims sourcés ou déclaratifs ? |
| Pratique | décisions concrètes ou théorie ? |
| Feedback | actionnable ou score seul ? |
| Vie privée | données minimisées ? |
| Gouvernance | revue contenu visible ? |
| Gamification | respectueuse ou compétitive ? |
| Différenciation | que retenir abstraitement ? |

## Familles à comparer

- académies fournisseurs IA ;
- cours en ligne généralistes ;
- modules corporate de sensibilisation ;
- guides institutionnels ;
- serious games sécurité/IA ;
- outils internes ou open source si disponibles.

## Patterns à adopter

- scénarios courts ;
- feedback immédiat ;
- progression par thèmes ;
- fiches réflexes ;
- mise à jour régulière ;
- langage accessible.

## Patterns à rejeter

- promesse de maîtrise par prompts ;
- score global nominatif ;
- source absente ;
- certification floue ;
- dépendance fournisseur non discutée ;
- collecte de données non expliquée.

## Fiche benchmark type

```yaml
name: Example AI Academy
source_url: https://...
accessed_at: YYYY-MM-DD
public_material_only: true
pedagogy:
  format: course|quiz|scenario|video
  feedback: none|basic|actionable
privacy_notes: ...
ideas_to_adapt_abstractly:
  - ...
ideas_to_reject:
  - ...
```

## Critères d'acceptation

- Chaque comparaison cite des sources publiques.
- Les limites juridiques sont respectées.
- Les idées reprises sont abstraites et originales.
- Aucune reproduction fidèle de contenu tiers.
