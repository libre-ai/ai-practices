# Corpus questions

Les fichiers de ce dossier sont des données pédagogiques versionnées.

## Règles

- `draft` : brouillon, non publiable.
- `review` : prêt à relire.
- `approved` : publiable seulement avec relecteurs, sources et audits médias.
- `blocked` : conservé comme trace, non utilisable.
- `retired` : retiré, conservé pour historique.

Le fichier `pilot.yml` contient des exemples de démarrage en statut `draft`, à relire avant publication.
